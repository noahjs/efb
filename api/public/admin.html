Airport Database

Downloads the FAA NASR 28-day subscription data and populates the airport, runway, and frequency tables. Currently loaded with sample Colorado airports. Running this will import all ~20,000 US airports.

Navigation Database

Downloads FAA NASR 28-day subscription data and populates the navaid (VOR, VORTAC, NDB, etc.) and fix/waypoint tables. Currently loaded with sample Colorado navaids and fixes. Running this will import all US navaids and named fixes.

Terminal Procedures (d-TPP)

Downloads the FAA d-TPP metafile XML and imports ~17,000 terminal procedure records (approach plates, SIDs, STARs, airport diagrams). PDFs are cached on-demand when viewed.

Aircraft Registry

Downloads the FAA Releasable Aircraft database and imports ~300K+ aircraft registrations with manufacturer, model, and engine data. Used for N-number lookup and auto-fill when creating aircraft.
